# Musiques de fond

Place ici des fichiers MP3 libres de droits, nommés selon le genre attendu par
le formulaire (`MusiqueAmbiante` dans `lib/types.ts`) :

- `jazz.mp3`
- `lofi.mp3`
- `dramatic.mp3`
- `engaging.mp3`
- `hyper.mp3`
- `inspirational.mp3`
- `documentary.mp3`

Le pipeline les mixe à 15 % de volume sous la voix off. Si le fichier
correspondant est absent, le montage se fait simplement sans musique.
